Quick Start: The easiest way is to use Python's built-in web server (see Option 1 below).

Option 1: Using Python (Recommended)

Python is pre-installed on macOS and Linux. Windows users can download it from python.org.

  1. Open terminal/command prompt in the dist folder
  2. Run this command: 
    python -m http.server 8000
  3. Open your browser to: http://localhost:8000
  4. To stop the server, press Ctrl+C in the terminal

Using the Application

  1. Click "Select Patient Folder"
  2. Choose the patient directory (containing patient ID and subdirectories)
  3. Select image set from the dropdown if multiple available
  4. Use tools to annotate images:
    • Select Tool: Mark capillary loops
    • Secondary Tool: Mark hemorrhages/avascular areas
    • ROI Tool: Set 1mm√ó1mm assessment box
    • Ruler: Measure features (set scale first)
  5. Click "Export Batch PDF" to generate comprehensive report
